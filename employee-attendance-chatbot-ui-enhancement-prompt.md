# AI Prompt: Employee Attendance System Chatbot UI Enhancement

## Context
You are an expert UI/UX designer and frontend developer specializing in modern web applications and chatbot interfaces. You have been provided with a UI specification document for an Employee Attendance System chatbot that includes current components, color palette, typography, and basic animations. Your task is to enhance this chatbot interface to make it more visually appealing, user-friendly, and effective.

## Current System Overview
The Employee Attendance System chatbot currently features:
- Basic chat interface with message bubbles
- Standard color palette and typography
- Basic fade-in animations for messages
- Simple form elements for attendance data entry
- Basic navigation and user flow

## Enhancement Requirements

### 1. Visual Transitions and Animations

#### **Micro-Interactions and Animations**
Please suggest detailed implementations for:

- **Message Bubble Animations**:
  - Smooth typing indicators with realistic timing
  - Staggered message appearance with spring physics
  - Hover effects on interactive elements
  - Message status indicators (sent, delivered, read)

- **Form Element Enhancements**:
  - Floating label animations for input fields
  - Progressive disclosure animations for multi-step forms
  - Smooth validation feedback with color transitions
  - Loading states with skeleton screens

- **Navigation Transitions**:
  - Smooth page transitions between different chatbot sections
  - Breadcrumb animations for complex workflows
  - Modal/overlay animations for detailed views
  - Parallax effects for visual depth

#### **Modern Animation Techniques**
Provide specific CSS/JavaScript animation recommendations including:
- CSS transforms and transitions with proper easing functions
- Framer Motion or similar library implementations
- GSAP timeline animations for complex sequences
- Intersection Observer API for scroll-triggered animations

- **Visual Accessibility**:
  - High contrast mode implementation

  - Progress indicators for multi-step processes
  - Error prevention and clear error messages
  - Simplified language options
